## Что это

GhostMode — клиентское приложение для VPN-сервиса GhostMode. Основано на форке [Hiddify](https://github.com/hiddify/hiddify-app).

Работает на Android, Windows, macOS и Linux.

## Возможности

- **VLESS + Reality** — трафик неотличим от HTTPS, DPI не видит VPN
- **Обход белых списков** — специальный режим, работающий даже при ограничениях интернета
- Автоматическое обновление подписки
- Тёмная и светлая тема
- Поддержка Android, Windows, macOS, Linux

## Начало работы

1. Скачай приложение под свою платформу
2. Зайди в [@GhostModeVPN_bot](https://t.me/GhostModeVPN_bot) в Telegram
3. Создай туннель и скопируй ссылку на подписку
4. Добавь подписку в приложение
5. Нажми «Подключить»

## Для пользователей в России

Если стандартный VPN не подключается из-за ограничений — используй конфиг **«Обход белых списков»** в приложении. Он появится автоматически после добавления подписки.
